# 牌の記述

m\*:マンズの*  
p\*:筒子の*  
s\*:ソウズの*  
(e,s,w,n):(東、南、西、北)  
haku:白  
hatsu:發  
chun:中  

---

*には1~9までの数字が入る



# API

- 形式
